# RMT（若梦兔）
**Read this in other languages: [English](README.en.md).**
#### 介绍
ahkv2语言实现的按键宏，鼠标宏，手柄宏，图片识别，颜色识别，文本识别等功能的工具

Forked from: https://github.com/zclucas/RMT

#### 环境配置（如果使用手柄宏）
1.  双击安装Joy目录下的vJoySetup.exe，安装失败请重新安装
2.  运行vJoyConf.exe（点击Win键，然后搜索vJoy，点击Configure Joy）
3.  修改界面的Number of Buttons值为32，然后点击apply
4.  双击运行此软件即可
